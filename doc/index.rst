Sistema de Tickets
==================

Este es el módulo **Sistema de Tickets** para Odoo, diseñado para gestionar solicitudes, problemas y tareas mediante la creación y seguimiento de tickets.

Características
---------------
- Gestión de tickets por categorías.
- Flujo de trabajo de estados (Nuevo, En Progreso, Resuelto, Cerrado, etc.).
- Historial de actividades con soporte para imágenes.
- Interfaz de usuario amigable.
